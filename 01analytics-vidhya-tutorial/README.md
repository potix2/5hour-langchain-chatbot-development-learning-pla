## 環境セットアップ

### 1. 仮想環境の作成とアクティベート

```bash
# 仮想環境の作成
python -m venv venv
# 仮想環境のアクティベート
source venv/bin/activate
```

### 2. 依存パッケージのインストール
仮想環境をアクティベートした状態で以下のコマンドを実行します：

```bash
bash
pip install -r requirements.txt
```

### 3. 環境変数の設定
`.env`ファイルにOpenAI APIキーが設定されていることを確認してください。


## 実行方法

仮想環境がアクティブな状態で以下のコマンドを実行します：

```bash
python main.py
```
